**background-image** - встановлює фонове зображення елемента або задає градієнтну заливку **Зображення НЕ встановлює розмірі об”єкта!**

Якщо ми вказуємо перед градієнтом в універсальній властивості background будь який колір, то він втрачає сенс, бо його перекриє градієнт.

background-repeat: repeat; - за замовченням. Зображення буде повторюватись n-кількість разів ( якщо блок більший ніж фото ), щоб заповнити весь блок, як по ширині, так і по висоті.

background-repeat: no-repeat; - зображення не буде повторюватися

**background-repeat: repeat-x;** - повтор по горизонталі

background-repeat: repeat-y; - повтор по вертикалі

**background-position** - задає початкове положення фонового зображення, де перше значення ( відсотки, рх, слово ) визначає розташування по горизонталі, друге - вертикалі, 0 0; - за замовченням ( або left top ; )

background-attachment - встановлює, чи буде прокручуватися або проскролюватися зображення разом із вмістом елемента ( блоком ).

**background-size: 50%; - тепер зображення буде підлаштовуватись під зміну, змінюється ширина блоку - змінюється ширина зображення.** **Якщо хочемо працювати від висоти, то ми повинні спочатку писати auto а потім вже висоту. ( якщо ми працюємо від ширини, то друге auto писати не обов”язково )**

background-size: contain; - масштабує зображення таким чином, щоб воно точно влізло в блок, по найменшій частині.

background-size: cover; - воно масштабує зображення таким чином, що це зображення повністю заполонило собою весь блок, якім би він не був. Але якусь частину зображення ми не зможемо побачити. Чудово працює з background-position: 50% ( center );
